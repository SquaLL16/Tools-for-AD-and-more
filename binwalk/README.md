# binwalk

## Overview

Binwalk is a fast, easy to use tool for analyzing, reverse engineering, and extracting firmware images.

## Installation

Clone the repo and:

    sudo python3 setup.py install


## Usage

Scan firmware

    binwalk firmware.bin

Extract files

    binwalk -e firmware.bin
